Режим передачи по имени, определенный в `dired.el':
Режим для "editing" списки каталогов.
в Dired вы "editing" список файлов в каталоге и (Необязательно) его подкаталоги в формате
`ls -lR'.  Каждый каталог - это страница: Использовать C-x [ and C-x ] Перемещать по
страницам. "Editing" означает, что вы можете запускать команды оболочки для файлов,
посещать, сжимать, загружать или скомпилировать байт, изменять их атрибуты файла
и вставить подкаталоги в один и тот же буфер.  Ты можешь "mark" файлы для последующих
команд или "flag" для их удаления, либо файл за файлом или по всем файлам, соответствующим
определенным критериям. Вы можете перемещаться с помощью обычных команд движения курсора.
Буфер доступен только для чтения. Цифры - это префиксные аргументы.
Жмите d пометить файл `D' для удаления.
Жмите m и Mark файл или подкаталог для последующих команд.
  Большинство команд работают с отмеченными файлами и используют текущий файл
если файлы не отмечены. Использовать числовой префиксный аргумент для работы
следующий ARG (или предыдущие -ARG if ARG<0) файлы или просто `1'
для работы только с текущим файлом. Префиксные аргументы переопределяют знаки.
Mark-С помощью команд выводится список отказов впоследствии.
Type ? Понять, почему что-то пошло не так.
Type u Снять отметку с файла или всех файлов вставленного подкаталога.
Type DEL Для резервного копирования одной строки и снятия отметки или unflag.
Type x Для удаления (eXecute) отмеченных файлов `D'.
Type RET to Find Файл текущей строки
  (Или поместил его в другой буфер, если это каталог).
Type o Найти файл или каталог Dired в другом окне.
Type i Вставить подкаталог в этот буфер.
Type R Переименовать файл или переместить отмеченные файлы в другой каталог.
Type C Копировать файлы.
Type s Для переключения Сортировка по имени / дате или изменению переключателей `ls '.
Type g Для чтения всех в настоящее время расширенных каталогов aGain.
  Это сохраняет все метки и скрывает субдиры снова, которые были скрыты ранее.
Use `SPC' или `DEL' Двигаться вниз и вверх по линиям.

Если Dired когда-нибудь запутается, вы можете либо набрать g, чтобы прочитать
Каталогов, введите l, чтобы relist файл в точке или отмеченные файлы или
Подкаталог, и type M-x dired-build-subdir-alist Для разбора буфера
Снова для дерева каталогов.
