README для справочного руководства Emacs Lisp.

* Этот каталог содержит исходные файлы texinfo для Emacs Lisp
Справочное руководство.

* Сообщайте об ошибках в Руководстве по Lisp (или в Emacs), используя
M-x report-emacs-bug.
Чтобы задать вопросы, используйте список рассылки help-gnu-emacs.

* Справочное руководство по Emacs Lisp довольно велико. Он составляет около
1100 страниц в формате записной книжки; Общее количество файлов информации
составляет около 3,0 мегабайт.

* Вы можете отформатировать это руководство для Info, для печати печатной копии с
помощью TeX,
Или для HTML.

* Вы можете купить красивые печатные копии из Free Software Foundation.
Покупка руководства из Free Software Foundation помогает поддерживать наш GNU
Разработка работа.  See <http://shop.fsf.org/>.
(На момент написания этого руководства это из печати.)

* Главный файл для форматирования этого руководства для Tex называется 'elisp.texi'.
Он содержит команды @include для включения всех разделов, которые составляют
руководство.

* Этот дистрибутив содержит Makefile, который вы можете использовать с GNU Make.

** Чтобы создать Info-файл, вам нужно установить Texinfo, а затем запустить
'make info'.

** Use 'make elisp.pdf' or 'make elisp.html' to create PDF or HTML versions.
