fopen
#include <stdio.h> /* Включая стандартную библиотеку */
//#include <windows.h> /* Раскомментировать это для Windows */
 
FILE * fopen (const char * restrict path, const char * restrict mode)
Открывает файл с именем restrict path и возвращает поток, или NULL при ошибке. Режим может
быть одним из следующих для файлов или одной из этих строк с включенным b (после первого
символа) для двоичных файлов.

Аргументы

Открыть файл
Аргументы для режима:
“r” Открыть файл для чтения
“w” Создание/запись
“a” Добавить 
“b” Открытие двоичного файла
“rb” Открыть двоичный файл для чтения
”+” Открыть файл для чтения/записи
“r+” Обновление текста (чтение и запись)
“w+” Обновление текста, отмена предыдущего содержания (Если таковые имеются)
“a+” Добавление текста, чтение и запись
примеры fopen(“test.txt”, “r+”);, fopen(“test.txt”, “w+”);, fopen(“test.txt”, “rw”); …
