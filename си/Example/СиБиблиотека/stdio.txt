stdio.h
Функция		Описание
BUFSIZ		(Целое значение) Размер буфера
clearerr	Удаляет индикаторы конца файла и ошибки для потока
fclose		Закрыть текущий поток / файл
feof		Проверяет наличие конца файла
ferror		Проверяет наличие потоковой ошибки
fflush		Синхронизирует выходной поток с фактическим файлом
fgetc		Читает байт / символ из потока файлов
fgetpos		Получает индикатор положения потока
fgets		Читает строку байта / символа из потока файлов.
fopen		Открыть поток
fprintf		Печать отформатированного байта / символа вывода в поток
fputc		Записывает байт/символ в поток
fputs		Записывает строку байта/символа в поток
fread		Читает X количество объектов из потока
freopen		Закрывает файл, связанный с потоком, затем открывает по имени файла в
		заданом	режиме и связывает его с потоком.
fscanf		Форматированное чтение со стопы
fseek		Переместите указатель позиции файла в определенное место в потоке
fsetpos		Переместите указатель позиции файла в определенное место в потоке
ftell		Возвращает индикатор текущей позиции файла
fwrite		Записывает X количество объектов из потока
getc		Читает байт/символ из потока
getchar		Читает байт/символ из stdin
gets		Читает строку байта из stdin
perror		Печатает текущее значение символа strerror (errno) в stderrr
printf		Печатает форматированный вывод на стандартный вывод
putc		Записывает байт/символ в поток
putchar		Записывает байт/символ в stdout
puts		Записывает строку байта/символа в stdout
remove		Удаляет файл
rename		Переименовать файл
rewind		Перемещает индикатор позиции файла в начало (OL) в файле
scanf		Чтение форматированного байта/символьного ввода из stdin
setbuf		Устанавливает буфер для потока
setvbuf		Устанавливает буфер для потока
sprintf		Печать отформатированного байта/символа до стандартного вывода
sscanf		Чтение форматированного байта/символьного ввода из stdin
tmpfile		Возвращает указатель на временный файл
tmpnam		Возвращает уникальное имя файла
ungetc		Помещает возврат байта/символа (push-back) в поток
vfprintf	Печатает форматированный вывод байтов/символов в поток с использованием
		переменных агументов
vprintf		Печатает отформатированный вывод байтов/символов в stdout с использованием
		переменных аргументов
vsprintf	Печатает форматированный вывод байтов/символов в поток с использованием
		переменных агументов
